# Bash Programming Tasks

## Описание
Этот репозиторий содержит решение заданий по Bash программированию. Каждый скрипт выполняет отдельную задачу и находится в соответствующем файле.

## Список скриптов

1. **greeting.sh** - Скрипт запрашивает имя пользователя и выводит приветствие.
2. **calculator.sh** - Калькулятор суммы двух чисел.
3. **even_odd_check.sh** - Проверка числа на четность.
4. **project_structure.sh** - Создание структуры папок для веб-проекта.
5. **line_counter.sh** - Подсчет количества строк в указанном файле.
6. **password_generator.sh** - Генератор случайного пароля длиной 8 символов.
7. **file_searcher.sh** - Поиск файлов по расширению в текущей директории.

## Требования
- Bash
- Базовые утилиты Linux/Unix (для некоторых скриптов)

## Инструкция по запуску
1. Склонируйте репозиторий
2. Перейдите в директорию проекта
3. Выдайте права на выполнение скриптам: `chmod +x *.sh`
4. Запустите нужный скрипт: `./имя_скрипта.sh`
